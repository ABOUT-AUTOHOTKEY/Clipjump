Clipjump

Avi Aryan


Settings.ini stores settings for the ClipJump Program.
It can be found in the directory of Clipjump.exe named as Settings.ini
It has 8 Parameters.
  • Main
    1. Minimum_No_Of_Clips_to_be_Active
      It is the minimum no of clipboards that you want simultaneously to be active.
      IF YOU WANT 20, SPECIFY 20.
      If you want Unlimited, leave it blank. (Not Recommended)

    2. Threshold
      Threshold is the extra number of clipboard that will be active other than your minimum limit..
      Most recommended value is 10.
      Tip - Keeping a Threshold value of 10 will give you exact number of Clipboards as specified by the Minimum_No_of_Clips parameter.
      3. Note - With a Minimum_No_Of_Clips_to_be_Active set as 20 and Threshold at 10, you will at times have a maximum 30 clipboards.

    3. Show_Copy_Message
      This value determines whether you want to see the "Transfered to ClipJump" message or not while copy/cut operations.
      1 = enabled (default)
      0 = disabled

    4. Quality_of_Thumbnail_Previews
      The quality of Thumbnail previews you want to have.
      Thumbnail preview's qulaity if set high can cause slow response from the program.Also this can cause more disk space consumption by the Cache Folder.
      So, it is recommended to let it be 20

    5. Keep_Session
      Should ClipJump keep all the saved clipboards after each Clipjump restart or simply Windows restart if you are running it at Start up.
      1 = Yes (Clipboards kept)
      0 = No

    6. Remove_Ending_Linefeeds
      Remove Linefeeds from end of Clips . These linefeeds if not removed can cause an Extra ENTER to be stimulated while pasting Clips in Text-Holders.
      1 = Yes (Recommended)
      0 = No

  • System
    1. Wait_Key
      This is a Clipjump behaviour parameter. If you face Duplicate entries while pasting data from Multiple Clipboards , increase this entry.
      Default = 200
      It is not recommended to edit this entry, If you have problems contact the Author

    2. Version
      The current version of Clipjump. Dont Touch !
