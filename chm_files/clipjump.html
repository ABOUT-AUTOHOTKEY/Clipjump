Clipjump

v 11.0

Clipjump is a Multiple-Clipboard management utility for Windows. It was built to make working with multiple clipboards super fast and super easy. The program records changes in the system clipboard, stores them without any limits and provides innovative ways to work with them.


Here are few reasons to like Clipjump.
  • It Captures every change on Clipboard regardless of data-types.
  • Supports preview of Clipboard stored images such as Printscreen captures.
  • Control Multiple-Clipboards only by using the single shortcut Ctrl+V (which is also configurable)
  • Can have as much as simultaneous clipboards as possible, no LIMIT.
  • Very (very) fast Clipboard switching.
  • Fast and powerful searching to help find needed content while pasting.
  • Batch/Multiple Pasting
  • Highly Configurable, thanks to ClipjumpCustom
  • Not touches the registry and System drive.
  • Multilingual, free, portable, Unicode, open source


Although Clipjump may look simple and a minimal Clipboard manager in the beginning, it has the power features to make it suit everybody needs.
  • Extend-able and customizable with Plugins
  • Channels - Have as many as multiple Clipboard channels in Clipjump that store data independent of each other. These channels can have distinct names and hold data without inconsistency.
  • History Tool - View, search and restore old clipboard items including images.
  • FIX Clipboards at certain positions via FIXATE option.
  • Customize Clipjump and create little macros with custom hotkeys with the customization feature.
  • Action Mode - One place to execute all other components of Clipjump. Keeps you away from the trouble of learning many shortcuts.
  • Add any clip's data to System clipboard if needed.
  • Hold Clip - Feature to copy clip to buffer and have it not recorded by Clipjump.
  • Export and share clipboards with your friends.
  • Clipjump Controller - Totally control Clipjump from a GUI and/or command-line switches .
  • Copy file path, folder path and the data stored in file to Clipboard from any file manager.
  • Custom Paste Formats - Paste formats like UPPERCASE, SentenceCase and No-Formatting available from default. You can create your own too.
