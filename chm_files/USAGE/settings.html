Settings Editor



Clipjump Settings Editor provides a visual interface to change the basic settings of Clipjump.
To get help on any setting, keep the mouse static for a while after moving the mouse pointer over it . A Tooltip with description text should pop-up.
After changing settings, hit Apply and then OK.


Tips

  • To disable a shortcut , set it to None . You can do this by focusing cursor in the Shortcut box and then tapping Backspace once.
  • It is recommended to let Quality of preview thumbnail be 90 .
  • Keep Threshold = 1 to have the Maximum number of active clipboards equal to the Minimum number of Active clipboards.
  • Avoid using Alt only shortcuts as they can cause conflicts with other applications.
