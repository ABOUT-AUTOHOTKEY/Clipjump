I see the error "invalid hotkey"
This error happens because the system doesn't have a particular key, most probably if you are on a non-english computer like Russian.
To fix it, change your system default language to english. Alternatively, it would be great if you contact the author .
Added This error should be fixed now. If it still persists at selective places, please report.
